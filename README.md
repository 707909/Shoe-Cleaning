## Features

- **Add Customer Data** – Input customer name, phone number, service type, price, date, and status.
- **Service Selection** – Choose available services (e.g., Regular, Deep Clean, Repaint).
- **Status Tracking** – Manage order status (Process, Completed, Picked Up, etc.).
- **Save Data** – Store order information into the table.
- **Order Table** – Displays customer name, phone number, service, price, date, status, and actions.
- **Edit & Delete** – Update or remove existing orders.

- HTML
